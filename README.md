# Portafolio Web - Nelly Ferrada

Portafolio personal desarrollado con Vue 3 y Vite para presentar mi perfil profesional, habilidades técnicas y proyectos desarrollados durante mi formación como Desarrolladora Front-End Trainee.

## Secciones

- Inicio
- Sobre mí

- Portafolio

Incluye la App del Clima del Módulo 8, desarrollada con Vue 3, Vue Router, Pinia, Open-Meteo API, favoritos, estadísticas, alertas meteorológicas y pruebas unitarias.

- Vite
- JavaScript
- CSS
- Git
- GitHub
